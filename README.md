# PRIMORDIAL

Physics God Game — Mobile (Android/iOS)
Godot 4.4+ | GDScript

## Setup
1. Apri `project.godot` in Godot 4.4+
2. Leggi `CLAUDE.md` prima di qualsiasi modifica
3. Gli autoload sono già configurati nel project.godot

## Struttura
Vedere CLAUDE.md sezione 2 per struttura completa.
